Commands

  • 8ball
    • Command Description: Magic 8ball will answer your question | Make sure you focus your intent when asking a question!​
    • Usage: /8ball question:your-question-here
    • Example: /8ball question:is cabot the best bot?
  • Bibliomancy
    • Command Description: Get a religious text script from a random or specific source​
    • Usage: /bibliomancy type:biblical/spiritual/random form:text/embed
    • Example 1: Biblical; /bibliomancy type:biblical form:text - Will send you a script from the bible in text form
      /bibliomancy type:biblical form:embed - Will send you a script from the bible in embed form
    • Example 2: Spiritual; /bibliomancy type:spiritual form:text - Will send you a script from a sprirtual book, source or quote in text form
      /bibliomancy type:spiritual form:embed - Will send you a script from a sprirtual book, source or quote in embed form
    • Example 3: Random; /bibliomancy type:random form:text - Will send you a random script in text form
      /bibliomancy type:random form:embed - Will send you a random script in embed form
